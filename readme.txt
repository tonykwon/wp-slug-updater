Requires at least: 3.0

License: GPLv3

WP Slug Updater allows you to change Post and Term slugs on 'title' change.

== Description ==

Are you replicating an already existing site and translating it into different languages with the full sitemap intact (say Multisite)? Did you ever wish you could simply change the post tile and have WordPress to automatically update the slugs?

With WP Slug Updater you can.

== Installation ==

Upload the WP Slug Updater plugin to your blog, Activate it.
